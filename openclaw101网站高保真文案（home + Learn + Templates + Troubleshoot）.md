**Preview copy:**  
Possible causes include incomplete channel pairing, a stopped daemon, missing permissions, or invalid model settings.

- Verify your channel pairing
    
- Confirm the daemon is running
    
- Test your model/API configuration
    
- Review recent logs

## 4.4 热门问题区

### EN Section Title

**Common fixes**

**Bot not responding after setup**  
Check channel pairing, daemon status, and whether requests are reaching the system.  
**CTA:** Open fix

**Skill installed but nothing happens**  
Verify installation, dependency assumptions, and whether the skill fits your current workflow.  
**CTA:** Open fix

**Model requests fail silently**  
Inspect API configuration, health checks, and recent logs before assuming the system is broken.  
**CTA:** Open fix

**Browser automation fails mid-run**  
Check permissions, environment assumptions, and whether the target page flow has changed.  
**CTA:** Open fix

### ZH 区块标题

**高频修复问题**

**完成配置后，机器人还是没回复**  
检查渠道配对、daemon 状态，以及请求是否真的进入了系统。  
**按钮：**打开修复方案

**技能装好了，但什么都没发生**  
验证安装是否成功、依赖前提是否满足，以及这个技能是否适合你当前工作流。  
**按钮：**打开修复方案

**模型请求悄悄失败**  
先检查 API 配置、健康检查和最近日志，不要一上来就认定系统坏了。  
**按钮：**打开修复方案

**浏览器自动化运行到一半失败**  
检查权限、运行环境前提，以及目标页面流程是否发生了变化。  
**按钮：**打开修复方案
