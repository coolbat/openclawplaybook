## What is stored locally

The current product surface stores only display preferences in the browser.

### Theme and language preferences

The site stores your selected theme and language in local browser storage so the interface does not reset on every page load.

## What is not collected

This statement reflects the current scope of the site.

### No user accounts

No registration, login, or personal profile system is currently provided.

### No built-in payments

The site does not currently process payments or store billing information.
