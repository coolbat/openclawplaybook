## When to use this template

Use this when meetings happen regularly but the value disappears after the call ends. This template helps teams keep decisions, tasks, and owners visible.

## Minimal setup

Feed it one raw note source and define one output format. The first goal is not perfect summarization. It is making sure action items and owners survive the meeting.

## What good output looks like

The recap should show decisions, action items, owners, and open questions. Someone who missed the meeting should still understand what changed.

## First extension

Add a follow-up reminder loop for overdue actions. That is usually where the biggest collaboration value appears.
