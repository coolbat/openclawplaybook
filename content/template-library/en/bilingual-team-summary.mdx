## When to use this template

Use this when one update needs to be shared across mixed-language teams. The goal is not perfect translation. The goal is keeping one team rhythm without splitting context.

## Minimal setup

Start with one update format and one shared destination. Keep the message structure stable so both language versions remain aligned.

## What good output looks like

The summary should preserve the same priorities, owners, and action items in both languages. Readers should not feel like one side received a reduced version.

## First extension

Add audience-specific sections only after the bilingual core is consistent. Otherwise the two versions drift apart too quickly.
