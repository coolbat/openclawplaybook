## When to use this template

Use this when you already have a recording or transcript and want a reusable format for publishing. It is especially useful when the bottleneck is not recording the episode, but packaging it well afterward.

## Minimal setup

Start with one transcript input and one publishing destination. Keep the requested output simple: title, short summary, key moments, and a few timestamps.

## What good output looks like

Strong show notes help someone decide whether to listen and where to jump in. The structure should be clear enough that an editor can polish it quickly without rebuilding it from scratch.

## First extension

Add a section for memorable quotes or actionable takeaways. That usually makes the notes much more reusable across newsletters and social posts.
